##

Iniciei o projeto pensando em aprender na prática uma nova linguagem de programação, no caso, PHP.

Surgiu a ideia de criar um sistema de cadastro simples, que pudesse ser utilizado em uma escola para pesquisar informações sobre os alunos.

O banco de dados utilizado é o MySql, e o XAMPP também é utilizado para simular o servidor rodando localmente.
